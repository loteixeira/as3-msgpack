Packageorg.msgpack
Classpublic class Worker
InheritanceWorker Object

Worker base class. Workers are assigned to factories where they are used to encode/decode message pack data. Each type of data uses a own worker. If you want to create a custom worker (for a custom type) you need to create a class which extends this class.

public function disassembly(source:IDataInput):*
